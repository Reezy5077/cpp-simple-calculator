# Simple Calculator (C++)

This is a basic calculator program written in C++.

## Features:
- Addition
- Subtraction
- Multiplication
- Division

## How it works:
The user enters two numbers and selects an operator.
The program calculates and shows the result.

## What I learned:
- switch statements
- user input handling
- basic arithmetic in C++
- error handling (division by zero)
